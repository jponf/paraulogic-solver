# paraulogic-solver

Simple solver for the paraulógic game. Based on a static knowledge base of catalan words.

## Run locally

You can run the project locally using a built-in HTTP server:

**Using Python:**
```bash
python3 -m http.server 8000
```

**Using Node.js (npx):**
```bash
npx serve
```

Then open http://localhost:8000 in your browser.

## Tools

The [tools/](tools/) directory contains Python scripts for crawling and updating the word database. See [tools/README.md](tools/README.md) for usage instructions.
